# Horizon Mobile App

This is the React Native (Expo) client for the Horizon messaging platform.

## Getting Started

1. Install dependencies:
   ```bash
   yarn install
   ```
2. Start the Expo development server:
   ```bash
   yarn dev
   ```
3. To run on Android:
   ```bash
   yarn android
   ```
4. To run on iOS:
   ```bash
   yarn ios
   ```

## Environment Variables

- Configure any required environment variables in a `.env` file at the root of this package if needed.

## Project Structure

- `src/screens/` - App screens
- `src/contexts/` - React contexts
- `src/services/` - API and socket services

## Linting & Formatting

- Run `yarn lint` to check code style.
- Run `yarn format` to auto-format code.
